JavaScript Fundamentals

A comprehensive guide to adding, managing, and writing efficient JavaScript code.

1. Adding JavaScript to a Web Page

Efficiently adding code and using Developer Tools are the first steps in web programming. Proper integration ensures your website remains fast and responsive.

Methods of Integration

  • External Files: The most efficient method is using an external .js file. This separates logic from HTML, making code easier to read, maintain, and cache.
  • Inline Code: You can also use the <script> tag directly within the HTML for small snippets.
  • Best Practice Placement: Place your script tags just before the closing </body> tag. This ensures the JavaScript is "non-blocking" and does not delay the rendering of visual HTML content.

Your web browser's DevTools include a JavaScript Console for real-time debugging. It allows you to log information or view errors (including line numbers) as your code runs.

// Example of logging to the console

2. Variables & Data Types

Variables act as storage containers for values. JavaScript is "loosely-typed," meaning you do not need to specify the data type (like string or number) when declaring a variable.

Declaring Variables

  • let: Used for variables where the value can change or be reassigned later (e.g., a user's score).
  • const: Used for variables that must not change after assignment (e.g., a tax rate or ID).
  • Naming Standards: Use camelCase (e.g., userFirstName). Names must be unique within their scope, cannot contain spaces or dashes, and are case-sensitive.

Common Data Types

  • String: Text enclosed in single or double quotes ("Hello").
  • Number: Integers (25) or floating-point decimals (9.99) without quotes.
  • Boolean: Logical values that are strictly true or false.
  • Null: A special value representing a deliberate "no value" or empty state.
Tip: You can use the typeof operator (e.g., typeof myVar) to reveal a variable's current data type.

3. Working with Strings

Strings are sequences of characters used for text. You can manipulate them using built-in methods to format or extract information.

Template Literals

Modern JavaScript allows you to mix variables and text cleanly using backticks (`) and the ${variable} syntax, known as interpolation.

const firstName = "Marie";
console.log(`Hi ${firstName}, welcome back!`); 
// Outputs: Hi Marie, welcome back!

Essential String Methods

String methods generally return a new string; they do not mutate the original value.

  • .length: A property that returns the count of characters in the string.
  • .toUpperCase() / .toLowerCase(): Converts the text case.
  • .trim(): Removes whitespace from the beginning and end of the string.
  • .slice(start, end): Extracts a specific part of the string.
  • .replace("old", "new"): Finds and replaces specific content.

4. Working with Numbers

Numeric variables handle arithmetic operations. Unlike strings, numbers do not use quotes.

Arithmetic Operators

Operator Description
+ / - Addition and Subtraction
* / / Multiplication and Division
++ / -- Increment and Decrement (increase or decrease by 1)
+= Compound Assignment (adds value to self)

Operator Precedence

JavaScript follows standard mathematical rules: division and multiplication are performed before addition and subtraction. To override this order, wrap operations in parentheses ().

let score = 10;
score += 5; // Adds 5 to score, result is 15

// Precedence example
let result = (5 + 2) * 3; // Result is 21, not 11
